КВД — Медицинская информационная система

Веб-приложение для автоматизации учета специализированных заболеваний кожно-венерологического диспансера.  
Разработано в рамках дипломного проекта как замена устаревшей desktop-системы на C#/MySQL.

1. Стек технологий

Backend:
- Java 25, Spring Boot
- Spring Security (аутентификация, ролевая модель)
- PostgreSQL
- Spring Data JPA / Hibernate

Frontend:
- HTML, CSS, JavaScript (без фреймворков)
- Fetch API

Интеграции:
- DaData API — автодополнение адресов
- Apache POI — экспорт в Excel

2. Функциональность

- Управление пациентами: поиск, карточка пациента, история болезней
- Учёт случаев заболеваний: создание, редактирование, удаление
- Справочники: диагнозы, типы лечения и другие (с разграничением по ролям)
- Управление пользователями (операторы / администраторы)
- Журнал аудита действий пользователей
- Дашборд со статистикой и графиком распределения диагнозов
- Экспорт списка пациентов в Excel
- Защита целостности данных (пациент с историей болезней не может быть удалён)

3. Архитектура

- REST API на Spring Boot
- Ролевая модель: оператор (чтение) / администратор (полный доступ)
- Валидация DTO на уровне контроллеров (@NotBlank, @NotNull, @Size)
- Глобальный обработчик ошибок (GlobalExceptionHandler)
- Аудит: фиксация имени пользователя в записях (денормализация для стабильности)
- Пакетные запросы для избежания N+1

4. Запуск

Требования:
- Java 25+
- PostgreSQL 18+

Настройка:
1. Создайте базу данных PostgreSQL
2. Укажите параметры подключения в application.properties:
  properties
spring.datasource.url=jdbc:postgresql://localhost:5432/kvd
spring.datasource.username=your_user
spring.datasource.password=your_password

3. Запустите приложение:
   bash
./mvnw spring-boot:run
